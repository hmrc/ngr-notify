
# ngr-notify

### Run locally:
* `cd` to the root of the project.
* `sbt run` - to run the application using the default application.conf (also used for unit and integration tests)  
  `sbt "run -Dconfig.resource=application-dev.conf"` - to run the application with the application-dev.conf, which connects to HIP in QA.
* Navigate to http://localhost:1515
